# _SAE Lego_

## Description

SAE Lego est un projet permettant de transformer une image en pavage Lego.
Le programme Java prend une image et la redimensionne en appliquant différentes méthodes d'interpolation.

## Contexte

Ce projet permet de visualiser des images sous forme de mosaïque Lego, utile pour les passionnés de Lego, la création artistique ou les projets pédagogiques sur le traitement d’image.

## Prérequis

- Java 17 ou supérieur
- Maven pour la compilation du projet

## Exécution

- Avant d'exécuter le programme, il est essentiel d'adapter les chemins des images dans le fichier java.fr.uge.lego.main.Main selon vos fichiers locaux.
- Vous pouvez appliquer une ou plusieurs méthodes d'interpolation personnalisées (minimum 1).

Une fois cela fait, vous pouvez lancer le programme.

## Fonctionnalités principales

- Trois types d’interpolation pour le redimensionnement
- Possibilité d’utiliser plusieurs types d’interpolation combinés pour améliorer la qualité de l’image redimensionnée
	
## Limitations / À noter

- La gestion du stock de briques n’est pas implémentée ; seul l’architecture est fonctionnels
- Certaines options de configuration restent manuelles (comme le choix de l’image dans le Main)
